## 📌 Overview

This study presents a comprehensive systematic literature mapping (SLM) of deep learning-based lane detection methods.

A total of **185 primary studies** were analyzed to identify trends, architectures, datasets, and challenges in the field.

Due to journal constraints on the number of references, the full list of primary studies is provided here as supplementary material.

## 🔍 Structure of the Dataset

Each study is indexed using an identifier:

- **C1 – C185**

These IDs are used consistently in the manuscript (e.g., Table A2) to ensure traceability between the analysis and the original studies.
